Berapa lama proses top up?

Proses top up biasanya memakan waktu 1-5 menit setelah pembayaran berhasil. Untuk pembayaran via transfer bank, proses bisa memakan waktu hingga 15 menit.

Bagaimana jika pembayaran gagal?

Jika pembayaran gagal, dana akan dikembalikan secara otomatis dalam 1-2 jam. Jika tidak, hubungi customer service kami untuk bantuan lebih lanjut.

Apakah data saya aman?

Ya, kami menggunakan enkripsi SSL untuk melindungi semua data transaksi dan informasi pribadi Anda. Kami tidak pernah membagikan data kepada pihak ketiga.

Bisa top up untuk server luar negeri?

Ya, kami mendukung top up untuk berbagai server termasuk Asia, Global, dan lainnya. Pastikan memilih server yang tepat saat melakukan top up.
